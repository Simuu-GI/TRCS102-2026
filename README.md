**Introduction to HTML and Basic CSS**

On the first day of the training, we were introduced to the basics of web development. The trainer explained how websites are created using HTML (HyperText Markup Language) and how CSS (Cascading Style Sheets) is used to enhance the appearance of web pages.

We learned the basic structure of an HTML document, including:

* `<!DOCTYPE html>`
* `<html>`
* `<head>`
* `<title>`
* `<body>`

The trainer demonstrated the use of several commonly used HTML tags, including:

* Headings (`<h1>` to `<h6>`)
* Paragraph (`<p>`)
* Line Break (`<br>`)
* Horizontal Rule (`<hr>`)
* Bold (`<b>`, `<strong>`)
* Italic (`<i>`, `<em>`)
* Underline (`<u>`)
* Images (`<img>`)
* Hyperlinks (`<a>`)
* Lists (`<ul>`, `<ol>`, `<li>`)
* Tables (`<table>`, `<tr>`, `<th>`, `<td>`)
* Forms (`<form>`, `<input>`, `<button>`)
* Divisions (`<div>`) and Spans (`<span>`)

After learning HTML, we were introduced to the basics of CSS. The trainer explained the purpose of CSS in designing attractive web pages and demonstrated different methods of applying CSS:

* Inline CSS
* Internal CSS
* External CSS

We also learned about:

* CSS Selectors
* Colors
* Backgrounds
* Fonts
* Text Alignment
* Margins
* Padding
* Borders

Finally, we created a simple web page using HTML and applied basic CSS styling to improve its appearance.


By the end of the first day, I was able to:

* Understand the structure of an HTML document.
* Use common HTML tags to create web page content.
* Insert images, links, tables, and forms.
* Apply CSS styles using inline, internal, and external methods.
* Customize fonts, colors, spacing, and borders using CSS.
* Develop a simple static webpage with basic styling.

The first day of training provided a strong foundation in web development. I gained practical knowledge of HTML and CSS, which will help me build more structured and visually appealing websites in the upcoming training sessions.
